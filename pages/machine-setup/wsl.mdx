# Windows Subsystem for Linux (WSL)

## How to install

```powershell
wsl --install
```

Microsoft instructions are good. See [them here](https://learn.microsoft.com/en-us/windows/wsl/install).

<Callout type="info">

**Make sure to use WSL 2**

Verify with

```powershell
wsl -l -v
```

See how to fix [here](https://learn.microsoft.com/en-us/windows/wsl/install#upgrade-version-from-wsl-1-to-wsl-2).

</Callout>
